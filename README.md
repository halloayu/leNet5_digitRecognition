# hdrPro

## 项目介绍
 
最近在写个小作业, tensorflow框架下, LeNet-5模型训练Mnist数据集实现手写数字识别。tonado搭建后端,websocket实现客户端与服务器端的双向传输, h5+css+js完成前端页面。
